# Algorithm

백준, 프로그래머스, SWEA 문제 풀이를 Java와 Python으로 정리한 알고리즘 저장소입니다. 문제별 폴더 안에 풀이 코드와 문제 설명 README가 함께 정리되어 있어, 풀이 이력을 플랫폼과 난이도 기준으로 확인할 수 있습니다.

## 주요 내용

- `백준`: Bronze, Silver, Gold, Platinum 난이도별 풀이
- `프로그래머스`: Level 0, 1, 2 풀이
- `SWEA`: D3 풀이
- 풀이 언어: Java, Python

## 폴더 구조

```text
.
├── 백준/
│   ├── Bronze/
│   ├── Silver/
│   ├── Gold/
│   └── Platinum/
├── 프로그래머스/
│   ├── 0/
│   ├── 1/
│   └── 2/
└── SWEA/
    └── D3/
```

## 풀이 방식

- 각 문제 폴더는 문제 번호와 제목을 포함한 이름으로 관리합니다.
- Python 풀이는 온라인 저지 입력 형식에 맞춰 표준 입력을 사용하는 파일이 많습니다.
- Java 풀이는 백준의 `Main` 클래스 또는 프로그래머스의 `Solution` 클래스 형태로 작성되어 있습니다.

## 실행 참고

별도의 프로젝트 빌드 파일은 없으므로 각 풀이 파일은 단독 실행 또는 온라인 저지 제출을 기준으로 다룹니다.

Python 파일은 Python 3으로 실행할 수 있습니다.

```bash
python3 "백준/Silver/1260. DFS와 BFS/DFS와 BFS.py"
```

Java 파일은 온라인 저지 제출용 클래스명에 맞춰 작성된 경우가 있어, 로컬 컴파일 시 파일명을 `Main.java` 또는 실행 환경에 맞는 이름으로 맞춰야 할 수 있습니다.
